This project explores trends in business licensing in the City of Vancouver using cloud-based data analytics. The dataset used covers the years 1997 to 2012 and includes information about business types, license status, fees paid, and more.

AWS Services Used: EC2, S3, Glue DataBrew, Glue Crawler, Glue Studio, Athena, CloudWatch, CloudTrail, KMS

ðŸ”„ Project Phases

Uploaded dataset from EC2 to Amazon S3. Configured encryption (SSE-KMS), versioning, and cross-bucket replication.

Used AWS Glue DataBrew to create a data profile, analyze missing values and schema structure.

Applied DataBrew transformations (remove columns, fill values, convert formats). Output: Cleaned CSV + Parquet.

Crawled cleaned datasets using AWS Glue Crawler to create a structured Data Catalog.

Used Glue Studio to create an ETL pipeline and Athena to query summarized trends.
